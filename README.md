# MyFastAPI - Система управления студентами

Простой и быстрый REST API для управления информацией о студентах, построенный с использованием современного фреймворка **FastAPI** на Python.

## 📋 Описание проекта

MyFastAPI - это веб-приложение, которое предоставляет API для работы с базой данных студентов. Проект демонстрирует основные возможности FastAPI, включая роутинг, параметры запросов, обработку ошибок и работу с JSON данными.

## 🚀 Функциональность

- ✅ Получение всех студентов
- ✅ Фильтрация студентов по курсу
- ✅ Детальный поиск студентов по курсу, специальности и году поступления
- ✅ Работа с JSON файлами для хранения данных
- ✅ Интерактивная документация API (Swagger UI)

## 🛠️ Технологический стек

- **Python 3.x** - язык программирования
- **FastAPI** - современный веб-фреймворк для создания API
- **Uvicorn** - ASGI сервер для запуска приложения
- **Requests** - библиотека для HTTP запросов
- **JSON** - формат хранения данных

## 📦 Требования

```
fastapi[all]
requests
```

Все зависимости указаны в файле `req.txt`

## 🔧 Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/your-username/MyFastAPI.git
cd MyFastAPI
```

### 2. Создайте виртуальное окружение
```bash
python -m venv .venv
```

### 3. Активируйте виртуальное окружение

**На Windows:**
```bash
.venv\Scripts\activate
```

**На macOS/Linux:**
```bash
source .venv/bin/activate
```

### 4. Установите зависимости
```bash
pip install -r req.txt
```

## 🚀 Запуск приложения

```bash
uvicorn app.main:app --reload
```

Приложение будет доступно по адресу: `http://127.0.0.1:8000`

### Доступные варианты запуска:

- Интерактивная документация (Swagger UI): `http://127.0.0.1:8000/docs`
- Альтернативная документация (ReDoc): `http://127.0.0.1:8000/redoc`

## 📡 API Эндпоинты

### 1. Корневая страница
```http
GET /
```

**Ответ:**
```json
{
  "message": "Привет!"
}
```

### 2. Получить всех студентов (опционально фильтр по курсу)
```http
GET /students?course=1
```

**Параметры:**
- `course` (опционально) - номер курса для фильтрации

**Ответ:**
```json
[
  {
    "id": 1,
    "name": "Иван Петров",
    "course": 1,
    "major": "Информатика",
    "enrollment_year": 2022
  },
  ...
]
```

### 3. Получить студентов по курсу с фильтрацией
```http
GET /students/1?major=Информатика&enrollment_year=2022
```

**Параметры пути:**
- `course` (обязательно) - номер курса

**Параметры запроса:**
- `major` (опционально) - специальность студента
- `enrollment_year` (опционально) - год поступления (по умолчанию 2018)

**Ответ:**
```json
[
  {
    "id": 1,
    "name": "Иван Петров",
    "course": 1,
    "major": "Информатика",
    "enrollment_year": 2022
  }
]
```


## 📊 Формат данных студента

```json
{
  "id": 1,
  "name": "Иван Петров",
  "course": 1,
  "major": "Информатика",
  "enrollment_year": 2022,
  "gpa": 4.5
}
```

**Поля:**
- `id` - уникальный идентификатор студента
- `name` - полное имя студента
- `course` - номер курса (1-4)
- `major` - специальность/направление обучения
- `enrollment_year` - год поступления
- `gpa` - средний балл (GPA)
